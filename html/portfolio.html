Clone NetFlix

Clone NetFlix is a project developed with HTML, CSS and JavaScript, which aims to replicate the interface and functionalities of the NetFlix video streaming platform. With him, the users can have an experience similar to the original, including navigating through the different titles, categories, search and playback of videos. It is an excellent example of how technologies front-end can be used to create interactive and functional projects.

Instagram home page

In this project, I used HTML, CSS and JavaScript to create a cloned version of the Instagram. The home page features the login screen, followed by the image feed and a new post submission form. With the application of responsiveness techniques, the design is compatible with different screen sizes, ensuring a great viewing experience. navigation on both desktops and mobile devices.
