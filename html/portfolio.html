Clone NetFlix

O Clone NetFlix é um projeto desenvolvido com HTML, CSS e JavaScript, que tem como objetivo replicar a interface e funcionalidades da plataforma de streaming de vídeo NetFlix. Com ele, os usuários podem ter uma experiência semelhante à original, incluindo a navegação pelos diferentes títulos, categorias, busca e reprodução de vídeos. É um excelente exemplo de como as tecnologias front-end podem ser utilizadas para criar projetos interativos e funcionais.

Pagina Inicial do Instragram

Neste projeto, utilizei HTML, CSS e JavaScript para criar uma versão clonada da interface do Instagram. A página inicial apresenta a tela de login, seguida do feed de imagens e um formulário de envio de novas postagens. Com a aplicação das técnicas de responsividade, o projeto é compatível com diferentes tamanhos de tela, garantindo uma ótima experiência de navegação tanto em desktops como em dispositivos móveis.
